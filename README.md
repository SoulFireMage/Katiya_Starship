# Katiya Starship

A Hugo website deployed to GitHub Pages at [soulfiremage.github.io/Katiya_Starship](https://soulfiremage.github.io/Katiya_Starship)

## Local Development

1. Clone the repository with submodules:
   ```bash
   git clone --recurse-submodules https://github.com/SoulFireMage/Katiya_Starship.git
   ```

2. Run Hugo locally:
   ```bash
   hugo server -D
   ```

3. Open http://localhost:1313/Katiya_Starship/ in your browser

## Deployment

The site automatically deploys to GitHub Pages when you push to the `main` branch.

## Theme

This site uses the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme.
